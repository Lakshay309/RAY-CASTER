2. Create a new empty project in Visual Studio
3. Add these source files to your project:

   * `main.cpp`
   * `raycaster.cpp`
   * `player.cpp`
   * `player.h`
4. Link SFML libraries:

   * `sfml-graphics`
   * `sfml-window`
   * `sfml-system`
5. Place `arial.ttf` in the working directory. In project settings â†’ set it to **"Copy if newer"**
6. Build and run (`F5`)
